# security-project

platform: flask+JS

## 运行：
    安装任意版本JDK
    
    使用pip
        pip install -r requirements.txt
        python main.py
    使用anaconda
        conda install --yes --file requirements.txt
        python main.py

## 文件结构： 
- /security-project  
    - *.py  
    - /templates  
        - mainpage.html  
    - /static  
        - /styles  
            - mainpage.css
